Gameplay:

• Progressive scoring (200/400/800/1600 for ghosts)

• Power pellet mechanics

Scoring & Persistence:

• Real-time score/lives/level HUD

Visual Polish:

• Directional sprites for Pac-Man
• Ghost animations and vulnerability states

• Scrolling high scores on splash screen (arcade-style)
• Game over screen with leaderboard

Controls:
• Arrow keys for movement
• WASD alternative controls
