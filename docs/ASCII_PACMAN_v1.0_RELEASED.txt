What's Included
═══════════════

Core Features:
✅ Complete Pac-Man gameplay
✅ 74x20 maze with 470 dots + 4 power pellets
✅ Four ghost AIs (Blinky, Pinky, Inky, Clyde)
✅ Progressive scoring (dots, pellets, ghosts)
✅ Lives system (3 lives, respawn mechanics)
✅ Level progression with difficulty scaling

Advanced Features:
✅ High score persistence (top 10 with names)
✅ Visual effects (score popups, death animations)
✅ Scrolling high scores on splash screen
✅ Dual control schemes (arrows + WASD)
✅ Game states (menu, playing, paused, game over)
✅ High score entry screen

Technical:
✅ Pure Python (stdlib only, no dependencies)
✅ Cross-platform (Linux, macOS, Windows)
✅ Professional package structure
✅ Console entry points (ascii-pacman, pacman)
✅ 84 comprehensive tests
✅ PyPI-ready setup
